# E-Commerce Sales Analytics Dashboard --- Dishu Daksh

## Overview

This project showcases an interactive **E‑Commerce Sales Analytics
Dashboard** built using **Microsoft Power BI**.\
The dashboard helps analyze sales performance, profit trends, product
categories, and regional distribution of sales.\
It is designed to provide clear, actionable insights for business
decision‑making.

## Objectives

-   Monitor **total sales, profit, orders, and discount values**
-   Analyze **sales trends across states**
-   Identify **top performing product categories and sub‑categories**
-   Understand **customer segments and shipping preferences**
-   Provide interactive filtering for deeper exploration

## Tools & Technologies

-   Microsoft Power BI Desktop
-   Data Visualization
-   Business Intelligence
-   Sample Superstore Dataset

## Dashboard Features

-   **KPI Cards** displaying Total Sales, Profit, Orders, and Discount
-   **Sales Trend Chart** to monitor sales distribution across states
-   **Sub‑Category Analysis** highlighting top selling products
-   **Category Comparison Chart** for Technology, Office Supplies, and
    Furniture
-   **Interactive Map Visualization** showing geographic sales
    distribution
-   **Customer Segment Pie Chart** for Consumer, Corporate, and Home
    Office segments
-   **Slicers (Filters)** for:
    -   Ship Mode
    -   Region
    -   Segment
    -   Category

## Key Insights

-   Technology category generates the highest sales revenue.
-   Consumer segment contributes the largest share of total sales.
-   Certain states contribute significantly more revenue than others.
-   Shipping modes influence order distribution patterns.

## How to Use the Dashboard

1.  Open the `.pbix` file using **Power BI Desktop**.
2.  Use the slicers on the right panel to filter data by region,
    segment, category, or shipping mode.
3.  Hover over visuals to view detailed data insights.
4.  Interact with charts and maps to explore sales trends.
